## Задание 2: Ручной счёт ROC_AUC
Классификатор выдал следующие прогнозируемые метки класса и вероятности принадлежности к классу "1". На основе полученных данных рассчитайте метрику ROC_AUC. Тезисно описать ход решения.


| Истинная метка класса | Порог классификации (0.6) | Оценка вероятности |
|:---------------------:|:-------------------------:|:------------------:|
| 1 | 1 | 0.95 |
| 0 | 1 | 0.90 |
| 1 | 1 | 0.85 |
| 0 | 1 | 0.80 |
| 1 | 1 | 0.75 |
| 1 | 1 | 0.70 |
| 1 | 1 | 0.65 |
| 1 | 1 | 0.60 |
| 0 | 0 | 0.55 |
| 0 | 0 | 0.50 |
| 0 | 0 | 0.45 |
| 1 | 0 | 0.40 |
| 0 | 0 | 0.35 |
| 0 | 0 | 0.30 |
| 0 | 0 | 0.25 |

  ### Решение

1. Взял тетрадь в клетку.
2. Нарисовал график, где по оси x - FPR, а по y - TPR.
3. Там где метка класса была 1 - проводил чёрточку вверх, где 0 -  чёрточку вправо.
4. В итоге получилась ROC-AUC кривая.
5. Positive - 7 штук, Negative -  8 штук. Максимальный ROC-AUC = 7 * 8 = 56(клеток) или равен 1.
6. Площадь под кривой, т.е наш ROC-AUC = (56 - 14) / 56 или 1 - 14/56. (14 -  число клеток над нашей кривой).
7. Ответ: 0.75.
